The Sankey diagram illustrates migration flows between Austria’s federal states. The width of each flow represents 
the number of people relocating from one federal state to another, making it possible to identify the strongest
migration connections and regional movement patterns.

The graph below shows the development of net migration for each federal state over time.
By clicking on a federal state in the legend, individual lines can be shown or hidden. This allows users to examine
individual federal states in detail and compare migration trends across multiple federal states.
